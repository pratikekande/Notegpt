[![Java](https://img.shields.io/badge/Java-ED8B00?style=flat&logo=java&logoColor=white)](https://www.java.com/)

**NOTEgpt** is an **AI-powered note-making application** built using **JavaFX and Firebase**, integrated with **Gemini AI API**.  
It allows users to ask questions, receive AI-generated answers, and **save those responses as personal notes** for future reference.
